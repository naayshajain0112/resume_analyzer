# BGV LinkedIn vs. CV Consistency Checker

An AI-powered Resume vs LinkedIn Verification System that compares candidate CVs with LinkedIn profiles using Gemini AI to detect inconsistencies, missing information, and profile mismatches, generating structured verification reports for recruiters and HR teams.

## Tech Stack
- **Frontend Framework**: React 18+ powered by [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Build Tool**: Vite

## Getting Started

### Prerequisites
Make sure you have Node.js installed (version 18+ recommended).

### Installation
1. Clone the repository and navigate to the project directory:
   ```bash
   cd resume_analyzer
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Build the application for production:
   ```bash
   npm run build
   ```

## Key Features
- **LinkedIn Profile Comparison**: Compare dynamic LinkedIn inputs against uploaded documents.
- **Interactive Document Upload**: Seamless drag-and-drop file uploader supporting PDF and DOCX files.
- **Real-Time Verification Progress**: Fluid micro-animations guiding recruiters through the analysis.
- **Premium HR Analytics Dashboard**: A sleek, glassmorphic consistency report showing match scores, discrepancy alerts, and detailed section-by-section comparison.
